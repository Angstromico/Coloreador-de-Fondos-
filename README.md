# Coloreador-de-Fondos-

Coloreador-de-Fondos es una pequeña aplicación web para cambiar el color de fondo de la página usando HTML, CSS y JavaScript.

![Captura de pantalla del proyecto](fondo.png)

## Descripción

- `Colorea Fondo/`: versión inicial del proyecto.
- `final/`: versión final con la implementación corregida.
- `fondo.png`: imagen local utilizada como vista previa en este README.

## Funcionalidades

- Selección de color con entrada hexadecimal.
- Cambia el fondo de la página en tiempo real.
- Código simple y fácil de entender para aprender JavaScript básico.

## Cómo usar

1. Abre `final/index.html` o `Colorea Fondo/index.html` en tu navegador.
2. Introduce un valor de color en formato hexadecimal (por ejemplo, `#ff7733`).
3. Observa cómo cambia el fondo inmediatamente.

## Enlaces

- Demo en línea: [https://cambiafondos.netlify.app/](https://cambiafondos.netlify.app/)

## Estructura del proyecto

- `index.html` - página principal.
- `styles.css` - estilos del proyecto.
- `app.js` / `hex.js` - lógica para cambiar el color de fondo.
